Guerra Escamilla Leonardo Cuando queremos que el usuario tenga aun más interacción con los elementos de una página, las funciones de drag and drop son una muy buena opción. Éstas, al permitir arrastrar elementos html (como imágenes), hacen que varias tareas sean más sencillas para el usuario como añadir productos al carrito en una página de compras virtual, e incluso pueden emplearse para la elaboración de juegos de navegador bastante interesantes. Aquí es donde vuelve a tomar protagonismo el lenguaje JavaScript (con ayuda de la API Drag and Drop) que, como ya hemos visto, es increíblemente útil en el desarrollo web de Front-End.
Garcia Rios Valeria Gracias a esta práctica, aprendimos a utilizar las funciones de arrastrar y soltar en una página web, las funciones "drag" y "drop" hace que los usuarios interactúen con los elementos de nuestra página, asi que ellos arrastren el elemento a cualquier parte permitida de la pantalla, facilita la interacción del usuario con un programa o aplicación de manera intuitiva para llevar elementos de un lugar a otro.
