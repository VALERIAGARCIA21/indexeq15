Parte 2: Etiquetas Utilizadas

Instrucciones: Describir las Etiquetas utilizadas en la parte 1.

TITULOS < H1 > Y < H3 >

Estas etiquetas < h1> - < h6 > se utilizan para definir los encabezados HTML el < h1> es para el tema Principal y el < h6> para el tema menos importante.

Estas etiquetas las utilize en la practica de manera que la h1 fuera para el tema principal, mientras que el h2 lo utilize para dar una breve explicacion de lo que trataria la practica y el h3 para los subtemas del tema.

< P >

Es para definir un parrafo, en los navegadores agregan automaticamente una sola linea en blanco antes y despues de esta etiqueta; En esta etiqueta la utilizamos para describir el tema ademas de que le realizamos un diseño en CSS,

< A >

Es para definir un hipervinculo, se utiliza para vincular de una pagina a otra, el atributo mas importante de este elemento es el href, que es el que indica el destino del enlace.

Se utilizo para enlazar entre las paginas html, anteriormente utilizadas.

< Head >

Es un contenedor de metadatos (datos sobre datos) y se colocaentre la etiqueta < html > y la etiqueta < body >. Nosotros lo utilizamos dos veces primero para darle vista ala pagina web y despues se utilizo para la etique de navegacion.

< DIV >

Esta etiqueta esta definida como una division o seccion en un documento HTML, ademas de que se usa como contenedor para elementos Html, que luego se estiliza con CSS o se manipila en JavaScrip, siempre y cuando este utilizado junto al atributo class o id.

Esta etiqueta se utilizo para separar los subtemas y la descripcion de cada subtema ademas de que tambien se utilizo para que el apartado de los videos que se encuentran en la pagina

< Meta >

El contenido de metadatos es cuando configura la presentacion o el comportamiento del resto del contenido, o establece la relacion del documento con otrosdocumentos o que transmite informacion "fuera de banda".

Esta etiqueta es utilizada para poder enlazar las imagenes o videos que se encuentran dentro d la carpeta y asi puedan aparecer en la pagina web, ademas de que en esa parte tambien nombramos el titulo que queremos que sea posible de encontrar, tambien de que le nombramos la etiqueta de estilo para darle un poco de diseño a la pagina web.

< Body >

Es para definir el cuerpo del documento, este elemento contiene todo lo que debe incluir un documento en HTML, como encabezado, parrafos, imagenes, etc; Por lo tanto se utilizo para que ahi se encuentre todo lo de la pagina web.

< Nav >

Este elemento representa una seccion de una pagina que enlaza con otras paginas o partes dentro de la pagina.Esta etiqueta de fine el conjunto de enlaces de navegacion

Esta etiqueta se utilizo para enlazar la Practica 5 con la Pagina Principal y tambien enlazarla con esta parte de 2 de la Practica 5.

< Iframe>

Se utiliza como tipo de marco en linea, aunque tambien se utilizapara incrustar un documento dentro del documento HTML actual, Este lo utilizamos para poder integrar los videos para la pagina Web.

< Table >

Es para definir una tabla en el HTML consiste de uno o mas elementos como < tr> que es para definir la tabla , < th> define un encabezado de la tabla y < td> define la celda de la tabla.

La utilizamos para poder integrar nuestras conclusiones individuales sobre lo aprendido en el tema.
